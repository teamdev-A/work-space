## どうしてテトリスを選んだか？
ビジネスにインパクトを与えることを考えたときテトリスは拡張機能でその部分をとらえやすく、チームの技術力と一致していました。
 
 
## このテトリスの特徴は？
リピーターを増やすためにはどのような技術を取り入れるべきなのかを念頭において開発しました。

## 機能要件

### スタートページ

### ゲームページ

## 注意事項
対応機種はPCを想定しております。 基本操作は十字キー、回転はスペースキーで行います。
