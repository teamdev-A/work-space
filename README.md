# チーム開発によるテトリス

## 概要
チーム開発でテトリスが遊べるアプリケーションを開発しました。この開発ではDOM操作の練習を目的としました。

開発で使用したDOM操作については、HTMLと連携したアニメーションのような動的な操作とゲーム音声の制御とイベントの処理であり、これによりユーザーエクスペリエンスが向上しています。

ユーザーの最高点を記録するためにローカルストレージを使用しておりユーザーがブラウザを閉じてもスコアが保存され次回アクセス時にデータが復元されます。またテトリミノの生成はMathオブジェクトを活用したrandomIdxメソッドで疑似乱数を生成しており、これがランダムなテトリミノを表現しゲームを興味深くします。ゲーム画面の描画にはHTML Canvasを使用して2Dコンテキストを描画しています。

## どうしてテトリスを選んだか？
もう一度クリックしてもらえる制作物をチームで作りたいと考えてチームの技術力を吟味した結果テトリスがマッチしていたためこの開発に挑みました。また、テトリスの開発に伴って基本的なGitの操作にも慣れるため積極的にマージやプルリクエストを行っています。
